# **App Name**: Devendra More - Portfolio

## Core Features:

- Dynamic Content Loading: Asynchronously load data to populate the portfolio, including work history, skills, and projects, to ensure faster initial page load times. Utilizes React's state management for dynamic updates.
- Interactive Skill Graph: Create an interactive graph or chart visualization of Devendra's skills, showing proficiency levels. Users can hover over skills to see more details or project examples.
- Project Showcase: A visually engaging showcase of Devendra's projects, complete with descriptions, tech stacks used, and links to live demos or GitHub repositories.
- AI-Powered Personalization Tool: Implement an AI-driven tool that adjusts the layout and content emphasis based on inferred visitor interests (e.g., recruiters vs. peers). The AI will act as a tool, deciding on the weight and order for elements like 'work history' and 'skills'.
- Contact Form: A simple contact form that allows visitors to send Devendra a message directly.

## Style Guidelines:

- Background color: Deep charcoal (#121212) to set a modern, sophisticated tone.
- Primary color: Electric purple (#9400D3) to bring focus to interactive elements and section headings, inspired by the creative coding aesthetic and connoting both artistry and precision.
- Accent color: Vivid turquoise (#00FFFF) as a contrasting accent for hover states and micro-interactions.
- Font: 'Space Grotesk' (sans-serif) for both headlines and body text. This font brings a clean, modern aesthetic suitable for a tech portfolio.
- Use minimalistic, line-based icons to represent skills, technologies, and contact methods.
- Employ a grid-based layout with ample white space to create a clean, organized user interface. Sections should be clearly defined, making the information architecture immediately apparent.
- Incorporate subtle, smooth animations on scroll, hover, and transition states to enhance user engagement without overwhelming the user. Consider using Framer Motion for these effects.
